经济地理学 · 科学家流动 · 区域创新

我的研究以经济地理学为基础，关注科学家如何在机构与城市之间流动，知识生产如何在空间中组织，以及区域创新体系如何与人才流动、科研机构和城市机会相互关联。研究中主要使用科学家履历、论文、机构和城市等纵向关系数据，并结合网络分析、事件层面的目的地选择模型和空间测度方法。

当前研究重点

我目前以全球生物医药顶尖科学家为主要经验对象，研究科学家职业流动与知识生产的空间组织。一条研究线把已观察到的城际流动理解为事件层面的目的地选择；另一条研究线构建科学家—机构—城市数据，用于分析流动网络、合作记录与区域创新条件，并在研究设计中区分这些不同概念。

代表性成果

迁移前论文地点暴露与生物医药顶尖科学家的目的地选择

工作论文，研究科学家后续选择的城市是否已经在其起点城市锚定的迁移前论文地点环境中具有可见性。

城市网络视角下全球生物医药顶尖科学家流动的空间结构与影响因素

已被《世界地理研究》录用，分析全球生物医药顶尖科学家城际流动网络及其流动联系形成和强度的相关因素。

Scopus 科研数据工具包

面向 Scopus 书目数据处理的公开 Python 工具包，使用合成测试数据、离线模拟接口和明确的数据边界展示可复现流程。

方法

纵向关系数据构建；科学家—机构—城市匹配；实体解析；社会网络与空间网络分析；自我网络分析；事件层面目的地选择的条件 Logit；网络项目中的 WERGM/ERGM；Python、R、Stata、GIS、NetworkX、Gephi 和 Ucinet。
